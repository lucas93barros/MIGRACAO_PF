# 📊 Análise de Migração no Espírito Santo

Este projeto apresenta uma análise exploratória de dados migratórios com base no sistema SISMIGRA (Polícia Federal), com foco no estado do Espírito Santo.

## 🎯 Objetivo
Analisar o comportamento migratório no ES, identificando padrões por nacionalidade e tipo de permanência.

## 🔎 Principais análises
- Fluxo migratório ao longo do tempo
- Nacionalidades com maior entrada
- Classificação do migrante (residente, temporário)
- Mapa do Brasil com destaque para o Espírito Santo

## 🛠️ Tecnologias utilizadas
- Python (Pandas, Matplotlib, GeoPandas)
- Quarto (relatórios)
- GitHub Pages (publicação)

## ⚙️ Como executar o projeto

Este projeto utiliza o gerenciador de dependências `uv`.

### 1. Criar ambiente virtual
uv venv

### 2. Instalar dependências
uv sync

### 3. Renderizar o relatório
quarto render notebook/relatorio.qmd
